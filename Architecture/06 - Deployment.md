## CI/CD  
  
Use GitHub Actions to:  
  
1. Run linting and tests on every push  
2. Build the frontend and backend  
3. Deploy automatically to Azure after successful checks  
  
---  
  
## Environments  
  
Recommended environments:  
  
### Development  
- Local frontend  
- Local backend  
- Local PostgreSQL or development database  
  
### Production  
- Azure Static Web Apps  
- Azure App Service  
- Azure Database for PostgreSQL  
- Azure Blob Storage
